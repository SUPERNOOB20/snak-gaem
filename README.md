* You can find the latest release [here](https://github.com/SUPERNOOB20/snak-gaem/releases/latest)
* You can find the source code [here](https://github.com/SUPERNOOB20/snak-gaem/tree/master)


## HOW TO PLAY

* Click a difficulty (easy/normal/hard) to get started...!
* `Q` pauses the game
* `R` resets your game when it's GAME OVER.
* `Esc` (escape) goes back to the previous menu (or exits the game, if you already are in the main menu...)

* `D` is for debugging (debugging means fixing coding issues, a.k.a bugs e.e).
